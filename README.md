# 🚗 Automobile Market Analysis

## 📌 Project Overview

This project focuses on analyzing automobile market data using Microsoft Excel. The goal is to extract meaningful insights related to pricing, vehicle features, fuel types, and market trends to support data-driven decision-making.

---

## 🎯 Objectives

* Understand the distribution of car prices across different brands
* Analyze the impact of features (fuel type, transmission, engine size, etc.) on pricing
* Identify trends and patterns in the automobile market
* Perform data cleaning and preprocessing for accurate analysis
* Create interactive dashboards for better visualization

---

## 🛠️ Tools & Technologies Used

* Microsoft Excel

  * Pivot Tables
  * Charts (Bar, Line, Pie)
  * Conditional Formatting
  * Data Cleaning Techniques
  * Basic Formulas & Functions (VLOOKUP, IF, COUNTIF, etc.)

---

## 📂 Dataset Description

The dataset includes information about various cars with attributes such as:

* Car Brand / Company
* Model Name
* Price
* Fuel Type (Petrol/Diesel/Electric)
* Transmission Type (Manual/Automatic)
* Engine Capacity
* Mileage
* Year of Manufacture

---

## 🔧 Data Cleaning Steps

* Removed duplicate records
* Handled missing/null values
* Standardized inconsistent data formats
* Converted data types where necessary
* Created new calculated columns for better analysis

---

## 📊 Key Analysis Performed

* Price distribution across different car brands
* Comparison of manual vs automatic cars
* Fuel type analysis and its impact on price
* Correlation between engine capacity and price
* Identification of high-performing brands/models

---

## 📈 Dashboard Features

* Interactive filters for brand and fuel type
* Visual comparison of prices
* Trend analysis using charts
* KPI indicators (Average Price, Count of Cars, etc.)

---

## 🔍 Key Insights

* Premium brands have significantly higher average prices
* Automatic cars tend to be more expensive than manual ones
* Diesel vehicles show better mileage trends in most cases
* Engine capacity has a strong correlation with pricing

---

4. Explore insights through charts and pivot tables
